# Cuphead

A Go/Ebitengine clone of the *Tutorial* level from the game Cuphead. (Abandoned project: Save/load system failed, only the *Tutorial* level is playable)

## Controls

| Action | Controls |
| :--- | :--- |
| Move Left | A |
| Move Right | D |
| Duck | S |
| Jump | Space |
| Shoot | Left |
| Dash | Up |
| Lock movement | Down |
| 8-way aiming | WASD |
| Drop down from platform | S + Space |

## Build

Enter the `go run .` command in the project directory to build and play.a
